Common Traps & Rules

summary() ≠ Mode
Returns Min, Q1, Median, Mean, Q3, Max. Mode is never included.
x == NA is always NA
Never use == to check NA. Always use is.na(x).
byrow=FALSE by default
matrix() fills column by column unless byrow=TRUE.
df = n1+n2-2
Degrees of freedom for two-sample t-test.One-sample: df = n-1
t-test is PARAMETRIC
Assumes normality. Wilcoxon is the non-parametric alternative.
sep="\t" not sep="tab"
Tab character in R strings is the escape \t.
Var(aX+b) = a²Var(X)
Constant b vanishes entirely. Only scaling a matters. b shifts, not spreads.
norm(u, "2") is Euclidean
c(4,1) → sqrt(4²+1²) = sqrt(17) ≈ 4.12
E[X] = 1/λ for Exponential
If f(x)=λe^(−λx), mean=1/λ. f(x)=5e^(−5x) → mean=0.2
Cov is NOT always positive
Can be negative (inverse relationship) or zero. Correlation normalises it to [-1,1].
oneway.test() not oneway.anova()
oneway.anova() does not exist. Use oneway.test() or aov().
Binomial uses PMF not PDF
PMF for discrete. PDF for continuous. Binomial is discrete → PMF.
R is 1-indexed
x[1] is the first element. Python uses x[0].
lst[[i]] vs lst[i]
[[i]] extracts the element. [i] returns a sub-list.
Poisson: mean = variance
If var >> mean, data is overdispersed → use negative binomial.
cbind → adds columns
Two length-3 vectors → matrix with 3 rows, 2 columns.
